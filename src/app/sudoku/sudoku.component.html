This is a demonstration of using CSPs to solve Sudoku. The backtracking algorithm finds all possible solutions. If you leave an open-ended grid, it will take a lot longer since there will be many solutions. For Sudoku problems, one solution is enough (early termination) but the backtracking search finds the optimal solution for problems and therefore we dont enable early termination.
